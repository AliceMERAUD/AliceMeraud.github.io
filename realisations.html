Mission 1 : Analyse des besoins métiers chez Safran

Mission 2 : Automatisation et optimisation de Jira avec ScriptRunner

  • Technologies utilisées : Jira, ScriptRunner, Groovy, Visual Studio Code.
  • Développements réalisés :

    • Optimisation des workflows existants afin de mieux refléter les processus métier.
    • Amélioration de la gestion des tickets pour un suivi plus clair et plus rapide.

Stage chez Phyling

Missions 1 : Refonte de Site Web pour PHYLING

Lors de mon stage chez PHYLING, j'ai réaliser une refonte complète du site web de l’entreprise pour améliorer l'expérience utilisateur et intégrer des fonctionnalités modernes.

Missions 2 : Code de capteur 4G pour PHYLING

Lors de mon stage chez PHYLING, j'ai développé un capteur 4G destiné à envoyer des informations via un réseau 4G vers un site web via une API. Ce projet avait pour objectif de permettre la collecte de données en temps réel et leur transmission sur une plateforme en ligne pour un suivi à distance.

  • Technologies utilisées : C++, Librairie 6700, Python.
  • Créations apportées :
    • API de communication : développement d'une interface API permettant l'échange de données entre un site existant et le capteur 4G. Cela permettait au capteur d'envoyer les informations collectées vers la plateforme web en temps réel.
    • Transmission des données via 4G : implémentation de la fonctionnalité permettant au capteur d'envoyer les données quand la fonction de « record » (enregistrement des données) était activée, utilisant le module 4G pour assurer une communication stable et continue.
