🎯 Objectif

Ce projet vise à alléger le temps nécessaire à la création de menus adaptés à la saison et à organiser facilement des activités pour les jeunes. Il a été conçu pour répondre à des besoins pratiques du quotidien.

🛠️ Fonctionnalités

🍽️ Gestion de menus

  • Création de menus saisonniers
  • Ajustement automatique des quantités selon le nombre de personnes
  • Organisation des repas sur une semaine complète
  • Génération de listes de courses triées (légumes, produits frais...)

🎲 Organisation d’activités

  • Filtrage par âge des jeunes, nombre de participants et accessoires nécessaires
  • Ajout de nouvelles activités personnalisées

📥 Ajout de contenu

Un système permet de proposer des recettes ou des activités via une messagerie. Comme je n’utilise pas de serveur, je trie manuellement les propositions avant de les intégrer à la base de données.

🔧 Technologies

  • Site statique hébergé sur GitHub Pages
  • HTML, CSS et JavaScript
  • Base de données locale (JSON ou SQLite)
  • Sans serveur (pas de backend)

📅 Avancement

Ce projet a été commencé pendant mes vacances. Il est en cours de développement. La base de données demande du temps pour être suffisamment complète, mais j’espère finaliser ce projet avant la fin de l’année.

💡 Compétences développées

  • Gestion de projet personnel
  • Développement web front-end
  • Structuration de données
  • Adaptation à des contraintes techniques (pas de serveur)
  • Amélioration de l’expérience utilisateur
