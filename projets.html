Projet 1 : Feur Nite - Jeu Multijoueur

Développement d’un jeu multijoueur inspiré de Fortnite, mettant en œuvre des fonctionnalités avancées de gestion réseau avec Unity et Fusion. Le projet visait à offrir une expérience immersive et interactive en temps réel pour les joueurs.

  • Technologies utilisées : Unity, Fusion, C#.
  • Fonctionnalités développées :
    • Gestion d’armes : système permettant à chaque joueur de commencer avec une arme, de la lâcher, la ramasser ou l’échanger avec d’autres joueurs.
    • Effets visuels : mise en place de particules visibles par tous les joueurs lorsqu'un projectile touche une cible.
    • Inventaire extensible : prend en charge des objets variés tels que des kits de soin, grenades, et armes.
    • GameManager : gestion centralisée du spawn des joueurs et distribution d’objets initiaux.

Projet 2 : Maths Calculus Adventure - Jeu Éducatif en Mathématiques

Développement d'un jeu éducatif multijoueur immersif, *Maths Calculus Adventure*, visant à rendre l'apprentissage des mathématiques interactif et amusant. Ce projet utilise des mécaniques de jeu pour enseigner les tables de multiplication, les multiples, et la résolution d'équations, tout en permettant aux joueurs de progresser à leur propre rythme dans un environnement captivant.

  • Technologies utilisées : Unity, C#, Git, Notion, Discord, YouTube.
  • Fonctionnalités développées :
    • Jeu immersif : chaque niveau est conçu pour enseigner un concept mathématique (tables de multiplication, multiples, équations) de manière ludique et interactive.
    • Progression personnalisée : les joueurs peuvent avancer à leur rythme, avec des niveaux progressivement plus difficiles adaptés à leurs compétences.
    • Interface simple et intuitive : commandes faciles à utiliser pour permettre aux joueurs de se concentrer sur l'apprentissage des mathématiques sans distraction.
    • Multiplateforme : le jeu est conçu pour fonctionner sur plusieurs plateformes (mobile, navigateur, PC), garantissant une expérience accessible pour tous les utilisateurs.
    • Gestion des ressources et du projet : utilisation de Notion pour organiser les tâches et de Git pour le partage et la gestion du code.

Projet 3 : Gestion de Commandes pour une Pizzeria

Création d'un système complet pour gérer les commandes, livraisons, et l'inventaire d'une pizzeria, comprenant deux applications et deux sites web reliés à une base de données centralisée.

  • Technologies utilisées : HTML, CSS, JavaScript, C++, SQL.
  • Fonctionnalités principales :
    • Application Pizzaïolo : suivi des commandes en temps réel avec mise à jour de la base de données.
    • Application Livreur : gestion des trajets avec algorithme d'optimisation selon disponibilité et adresses.
    • Sites Web : interface client pour les commandes en ligne, et plateforme gestionnaire pour surveiller l'inventaire et analyser les données.

Projet 4 : Analyse des Données Météo et de l'Indice Humain

Développement d’un site web interactif permettant d'analyser la corrélation entre les conditions météorologiques et l’indice humain grâce à des données RDF. Ce projet combinait des visualisations dynamiques et de l'analyse de base de données.

  • Technologies utilisées : HTML, CSS, JavaScript, JSON, CSV.
  • Fonctionnalités :
    • Chargement de données RDF : température, précipitation, durée d’ensoleillement.
    • Calculs automatisés : affichage des résultats sous forme de tableaux et graphiques dynamiques.
    • Interface utilisateur intuitive pour effectuer des requêtes spécifiques.
